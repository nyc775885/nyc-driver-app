## 初次使用

### 第 1 步：添加车辆

打开 app → 菜单 ☰ → 车辆基本信息 → 填写：

- **品牌**：从下拉选（Tesla、Toyota、Honda 等）
- **车型**：选品牌后自动出下拉（如 Tesla 选 Model Y）
- **车牌**
- **TLC 牌照**（NYC 网约车必填）
- **VIN**（17 位车架号）
- **年份 / 颜色**

> 💡 车辆只是元数据，**不影响金额计算**。换车不会丢历史数据。

### 第 2 步：连接 Google Drive（推荐）

菜单 → 备份 & 同步 → 登录 Google → 数据自动同步多设备。

### 第 3 步：导入历史数据（可选）

如果你以前用 Fuelio：
- 在 Fuelio app 导出当月 PDF 报告
- 在本 app 支出页 → 📥 选 Fuelio PDF
- 选文件 → 全选 → 导入

如果你以前手记账：
- 直接每月 / 每周开始记新数据即可

## 日常使用

### 每天记零碎花费

仪表盘底部 ➕ 加号，弹出快捷按钮：

| 按钮 | 用途 |
|---|---|
| ⚡ 充电 | 输入金额、kWh、地点 |
| ☕ 咖啡 | 输入金额（默认 $4.67 可改）|
| 💵 现金小费 | 当天的现金小费 |
| 🅿 停车 | 停车费 |
| 🌉 过桥 | EZpass 实付（手动加每次过桥）|

### 每周（可选）

如果想看每周表现，粘贴 Uber 周报：

收入页 → ➕ 周记录 → 📋 粘贴 Uber 周报 → 在 Uber app 复制周报全文 → 粘贴 → 保存

> ⚠ **周报只是参考**，不进入计算。计算用月度账单为准。

## 每月必做

### 月底或下月初：粘贴 Uber 月度账单

这是**最重要的一步**——计算的真实数据来源。

**怎么找 Uber 月度账单：**
1. 打开 Uber Driver app
2. 收入 → Tax Summary（或税务汇总）
3. 选月份 → 复制全文（或下载 PDF 后用文本提取）

**粘贴步骤：**
1. 本 app → 收入页 → ➕ 添加月度账单
2. 📋 粘贴 Uber 月报 → 粘贴文本
3. 检查解析的字段：
   - 营业额 (grossFare)
   - 小费 (tips)
   - 奖励 (bonus)
   - 过桥退款 (tollReimbursed)
   - 平台抽成 (platformFee)
   - 趟数 / 里程
4. 保存

**会发生什么：**

| 情况 | App 反应 |
|---|---|
| 这个月还没存过 | 直接保存 ✓ |
| 这个月已有月报 | 弹"覆盖？" → 你确认 |
| 这个月已有手动加的过桥/平台费 | 弹三选一：①保留 ②删除已有 ③取消 |

---

### 月底也建议：

**把 EZpass 月度对账单的过桥费手动加进去**

- 支出页 → ➕ → 类别"🌉 过桥费(月结)" → 金额 = EZpass 月对账单总额 → 日期任选月底
- **这是支出，对应 Uber 月报里的过桥退款（收入）**
- 净效应 = $0（钱进出抵消），但报税时支出/收入都要列

> 如果你不想分两笔，直接在月报里把过桥退款字段清零，就当没发生过。报税时少算 $168，但净利润不变。

## 报税

### 月底 / 年底导出报税包

菜单 → 报告 → 完整报税包 → 选年份 → 生成 PDF

**包含：**
- 1099-K 数字（收入）
- Schedule C 支出明细（按类别）
- 月度营业额 / 利润趋势
- 估算自雇税 / 联邦税 / 州税
- 司机 / 车辆信息

**用法：** 直接发给会计师。

### 自己看的报告

菜单 → 报告 → 月度 / 年度

显示：
- 净利润
- 利润率
- 时薪
- 各类支出占比饼图
- vs 上月对比

## 常见问题

### Q: 我换车了，旧车的支出会消失吗？
A: **不会**。车辆只是标签，所有支出按月份汇总，跟车辆 ID 无关。新增新车 → 旧数据照样看得见。

### Q: 我导入了 Fuelio，又粘贴 Uber 月报，过桥费会被算两次吗？
A: **不会**。粘贴月报时会检测重叠，弹三选一让你决定。

### Q: 周报粘贴会不会和月报冲突？
A: **不会**。周报只存到 wl 数组（仅参考），**不进入金额计算**。月报才是真账。

### Q: 我手动加了一笔 Uber 过桥费 $13.10，月底粘贴月报怎么办？
A: 月报粘贴时会检测，弹：
```
⚠ 2026-01 已有以下支出：过桥费 1 笔 $13.10
本月账单内含：过桥退款 $168.45

如何处理？
  1 = 都保留
  2 = 删除已有，只用月度账单
  3 = 取消
```
建议选 **2**（删手动那笔，让月报作准）。

### Q: 数据丢失了怎么办？
A: 多重保护：
- localStorage 即时保存（不会丢）
- Google Drive 每次操作自动同步
- IndexedDB 每 60 秒自动快照
- 删除支持 10 秒撤销
- 菜单 → 备份 → 导出 JSON 备份（手动备份）

### Q: 多设备数据怎么同步？
A: 都登录同一 Google 账号。一台改了 → 几秒内同步到另一台。

### Q: 怎么完全清空所有数据从头开始？
A: 菜单 → 备份 → 🗑 清空所有数据。会要求确认两次。Google Drive 上的也会清。

### Q: 版本号是什么意思？
A: 比如 v3.11.0：
- 主版本 3：不同版本之间数据不一定兼容
- 副版本 11：新增功能
- 修订 0：bug 修复
- **跳过含 4 的数字**（如 .4 → .5）

## 快速参考

### 录入顺序（推荐）

```
日常 ──→ 仪表盘 ➕ 快捷按钮（咖啡/充电/小费）
   │
每周 ──→（可选）粘贴 Uber 周报作参考
   │
每月 ──→ 月底必做：
       ① 粘贴 Uber 月度账单
       ② 加 EZpass 月度过桥支出
       ③ 检查报告页的净利润
   │
每年 ──→ 1 月：导出完整报税包给会计师
```

### 数据来源 vs 用途

```
月度账单（sl）→ 收入计算 ✓ 报税 ✓
周报（wl）   → 仅看趋势，不算钱
日记账（dl） → 当天实时录入
支出（el）   → 所有花费
```
