# NYC Driver Tracker

NYC 网约车/出租车司机的收入支出管理 PWA。

**当前版本：v3.11.0**

---

## 核心理念

### 1. 车辆 = 工具
车辆只是元数据（车牌、VIN、品牌型号），**不影响任何金额计算**。换车、新增车不会影响历史数据。

### 2. 收入端：单一真实源
- **月度账单（sl）** = 收入的唯一真实数据源
- **周报（wl）** = 仅供参考，不进入计算
- **日记账（dl）** = 实时录入（出租车/补充用）

### 3. 支出端：手动+导入
- 手动添加（仪表盘快捷按钮）
- Fuelio PDF 一次性导入历史
- 都存到 `el` 数组

### 4. 平台抽成（platformFee）
- 月度账单里直接含平台费字段
- 计算时自动从收入里扣
- 避免 double count：手动加 platform 类别支出**不会**被算（已存在月度账单字段里）

## 数据流

```
┌─────────────────────────────────────────┐
│ 收入                                      │
├─────────────────────────────────────────┤
│ 月度账单粘贴 (Uber Tax Summary PDF)       │
│   ↓                                      │
│   sl 数组：grossFare, tips, bonus,        │
│           tollReimbursed, platformFee     │
│   ↑ 单一真实源                            │
│                                          │
│ 周报粘贴 (Uber Weekly Statement)         │
│   ↓                                      │
│   wl 数组（仅参考）                       │
│                                          │
│ 日记账（出租车/网约车）                    │
│   ↓                                      │
│   dl 数组                                 │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 支出                                      │
├─────────────────────────────────────────┤
│ 手动添加 / Fuelio 导入 / 仪表盘快捷       │
│   ↓                                      │
│   el 数组                                 │
└─────────────────────────────────────────┘

仪表盘 / 报告 → 自动按月份汇总
（不按 vehicleId 过滤）
```

---

## 计算公式

```
总营业额 = grossFare + tips + bonus + tollReimbursed + otherIncome
总支出 = el 里所有 + 固定月费 + 日记账 lease
       (排除 platform 类别避免 double count)
平台费 = sl.platformFee + dl.platformFee
净利润 = 总营业额 - 总支出 - 平台费
```

## 冲突检测

| 操作 | 检测什么 | 提示形式 |
|---|---|---|
| 手动加月结类别支出（保险/手机费等）| 同月同类别已存在 | ⚠ 警告 + 总额 |
| 手动加日常类别（咖啡/充电等）| - | 不警告 |
| 粘贴周报 | 同周同平台已存在 | ⚠ 覆盖确认 |
| 粘贴月报 sl | 同月同平台已存在 | ⚠ 覆盖确认 |
| 粘贴月报：检测 toll/platform 重叠 | 当月已有手动 toll/platform 支出 | ⚠ 三选一（保留/删除/取消）|
| Fuelio PDF 导入 | 月结类别冲突 | ⚠ 列出全部冲突让选择 |

---

## 删除操作

**所有删除统一行为：**
- 弹自定义对话框（标题 + 说明 + 红色"删除"按钮）
- 不用浏览器原生 confirm()
- 删除后**自动 10 秒撤销**banner（点击恢复）

---

## 多设备同步

- localStorage 即时保存（无 debounce 延迟，避免后台时数据丢失）
- 自动同步到 Google Drive
- Tombstone 机制：一台设备清空 → 通知其他设备清空

---

## 版本号规则

- **跳过含数字 4 的版本**（如 v3.11.4 → v3.11.5，v3.14 → v3.15）
- patch 部分保持 ≤ 2 位数
- 到 .99 就升 minor（v3.11.99 → v3.12.0）

## 主要功能

### 仪表盘
- 当月/全年切换
- 总营业额 / 净利润 / 时薪 / vs 上月
- 支出分布饼图
- 快捷按钮：⚡ 充电 / ☕ 咖啡 / 💵 现金小费 / 🅿 停车

### 收入
- Uber 月度账单 PDF 粘贴解析
- Uber 周报 PDF 粘贴解析
- 手动加月度账单 / 周记录 / 日记账
- 多平台支持

### 支出
- 按类别筛选
- 月份切换
- Fuelio PDF 批量导入
- "🔗 关联无车辆支出"工具

### 报告
- 月度/年度报告
- Schedule C 报税明细
- 完整报税包 PDF（一份给会计师）

### 备份 & 同步
- Google Drive 自动同步
- 导出 JSON / Excel
- IndexedDB 自动快照（每 60 秒）

---

## 已知限制

- 不支持 ES6 计算属性名（会崩溃，环境兼容性问题）
- localStorage 限制 ~5MB（一般够用）
- Service Worker 缓存可能滞后，强制刷新（Ctrl+Shift+R）解决
